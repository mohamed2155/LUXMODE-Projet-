# LUXMODE-Projet-
Ce projet présente une solution complète d'optimisation de la Supply Chain pour LUXMODE, une entreprise fictive de prêt-à-porter. L'objectif est de piloter l'ensemble du flux logistique, de la prévision de la demande et l'optimisation des stocks.

Le projet démontre une maîtrise de la Data Stack Supply Chain : Excel pour l'analyse, SQL pour la gestion de base de données, et Power BI pour le pilotage stratégique.
